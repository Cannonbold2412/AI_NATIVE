## Inputs
- `{{user_email}}`: Enter user email.
- `{{user_password}}`: Enter user password. Keep this value secure.
- `{{db_name}}`: Enter db name.

## Steps
1. Enter {{user_email}}
2. Enter {{user_password}}
3. Click "Sign in"
4. Wheel scroll (Δy=201)
5. Select {{db_name}}
6. Wheel scroll (Δy=-201)
7. Delete Database

9. Delete Database
