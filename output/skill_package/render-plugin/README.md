# Render Plugin

Automation plugin for Render.

## Skills

### `generated_skill`

Delete a database on Render dashboard

- Execution: `skills/generated_skill/execution.json`
- Inputs: `skills/generated_skill/input.json`
- Recovery: `skills/generated_skill/recovery.json`

## Usage

1. Read this README or `orchestration/index.md` to understand available skills
2. Use `orchestration/planner.md` to plan skill sequences
3. Execute using `execution/executor.js`
