# Render Plugin

Automation plugin for Render.

## Skills

### `delete_database`

Delete a database on Render dashboard

- Execution: `skills/delete_database/execution.json`
- Inputs: `skills/delete_database/input.json`
- Recovery: `skills/delete_database/recovery.json`

### `login_in_render`

Log in to Render dashboard

- Execution: `skills/login_in_render/execution.json`
- Inputs: `skills/login_in_render/input.json`
- Recovery: `skills/login_in_render/recovery.json`

## Usage

1. Read this README or `orchestration/index.md` to understand available skills
2. Use `orchestration/planner.md` to plan skill sequences
3. Execute using `execution/executor.js`
